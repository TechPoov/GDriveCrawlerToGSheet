<h1><strong>GDriveCrawlerToGSheet</strong></h1>


&nbsp;   <br>
![GitHub all releases](https://img.shields.io/github/downloads/TechPoov/GDriveCrawlerToGSheet/total.svg)

<p><em>A simple Google Apps Script tool that scans your Google Drive and lists every folder and file into a clean, clickable Google Sheet.</em>

</p>

<p></p>

<hr>

<h2><strong>1. Overview</strong></h2>

<p>Google Drive looks organized… until you try to actually <em>find</em> something. Deep folders, duplicate files, confusing names, and scattered links — it all piles up.</p>

<p><strong>GDriveCrawlerToGSheet</strong> gives you instant clarity.

&nbsp;   <br>It scans your Drive and creates a clean inventory with <strong>one-click hyperlinks</strong> for every item.</p>

<p>No copying, no moving, no changes.

&nbsp;   <br>Just visibility.</p>

<p></p>

<hr>

<h2><strong>2. What This Tool Does</strong></h2>

<ul>

&nbsp;   <li>

&nbsp;       <p>Scans Google Drive using Apps Script</p>

&nbsp;   </li>

&nbsp;   <li>

&nbsp;       <p>Lists <strong>all folders and files</strong> into a Google Sheet</p>

&nbsp;   </li>

&nbsp;   <li>

&nbsp;       <p>Adds <strong>hyperlinks</strong> for quick access</p>

&nbsp;   </li>

&nbsp;   <li>

&nbsp;       <p>Captures path, owner, timestamps, and more</p>

&nbsp;   </li>

&nbsp;   <li>

&nbsp;       <p>Lets you add unlimited custom columns</p>

&nbsp;   </li>

&nbsp;   <li>

&nbsp;       <p><strong>Preserves your custom columns on re-scan<br></strong> After the first scan, the sheet becomes your own tracking space. Add comments, tags, status, or notes — they stay preserved on every re-scan as long as the item still exists.</p>

&nbsp;   </li>

&nbsp;   <li>

&nbsp;       <p>Works with standard Google Sheets features (sort, filter, notes, color coding)

&nbsp;           <br>

&nbsp;           <br>

&nbsp;       </p>

&nbsp;   </li>

</ul>

<p>It <strong>does not</strong>:</p>

<ul>

&nbsp;   <li>

&nbsp;       <p>Copy files</p>

&nbsp;   </li>

&nbsp;   <li>

&nbsp;       <p>Move files</p>

&nbsp;   </li>

&nbsp;   <li>

&nbsp;       <p>Modify your Drive

&nbsp;           <br>

&nbsp;           <br>

&nbsp;       </p>

&nbsp;   </li>

</ul>

<p>Just a smart index of everything you have.</p>

<p></p>

<hr>

<h2><strong>3. Key Features</strong></h2>

<ul>

&nbsp;   <li>

&nbsp;       <p>✔️ Full Google Drive folder + file inventory</p>

&nbsp;   </li>

&nbsp;   <li>

&nbsp;       <p>✔️ One-click hyperlinks for every item</p>

&nbsp;   </li>
